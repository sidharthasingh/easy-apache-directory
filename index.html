Easy Apache v1.0.0

It's good to know :

  • This programs uses Text-Parsing to enable a UI-logic layer on the responses obtained from the apache server which returns the directory listing if an "index" file in not found.
  • Currently, this app requires to be ran on a server itself, but future version roll outs will solve this issue.
  • To be able to make AJAX calls to other domains, the HEADER setting must be changed in server's '.conf' file. Find the instructions here.
  • Share this link if you find this app useful.
